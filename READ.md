HealthTrack is a personal wellness tracking web application built with React.

---

##  Features
-  Home Page with quick start button  
-  About Page  
-  Features / Services Page  
-   Activity Tracker (Dynamic page using React Hooks & State)  
-  Contact Page with form handling  
-  Fully responsive design  
-  Built with React + React Router  
-  Deployed on Netlify  

---

##  Technologies Used
- **ReactJS**
- **React Router**
- **Netlify**
